Springboot部署

Kendall Cai   in Amoy
2021-11-10
  • Springboot

  1. 先使用Mavan install将项目打包成jar包

  2. 将target目录下的jar包拷贝到服务器

    scp xxx.jar username:ip/home/path/xxx.jar

  3. 将jar包放到任意目录，执行下面的命令

    
// 非后台运行模式
java -jar project.jar

  4. 后台运行

    //这种方法会把日志文件输入到你指定的文件中，没有则会自动创建。进程会在后台运行。
nohup java -jar project.jar >temp.text &

// 指定启动环境采用生产环境文件
nohup java -jar project.jar --spring.profiles.active=prod &
// 默认启动日志输出到当前目录下 `nohup.out`
