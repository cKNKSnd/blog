Orange Pi Airplay

Kendall Cai   in Amoy
2021-12-01
  • Pi

手头上有一个闲置的 Orange Pi Zero 2，刚好又发现了一个有趣的项目shairport-sync，可以将树莓派等设备设置为Airplay客户端。

将Orange Pi使用3.5mm音频线连接至功放，实现Airplay无线听歌。

# 安装

系统环境 Ubuntu Server 20

sudo apt install shairport-sync

# 配置

修改配置文件 sudo vim /etc/shairport-sync.conf，核心配置如下

alsa = 
{
	output_device = "hw:2";
	mixer_control_name = "PCM";
	mixer_device = "hw:2";
	output_format = "S16"; // 可选
}
  • output_device 指定输出的设备名，使用aplay -l查看
  • mixer_control_name 音频控制
  • output_format 可选配置，我这里配置为S16，否则输出的音频白噪严重。如果碰到白噪严重的情况，可多尝试几个参数。

# 重启服务

sudo systemctl restart shairport-sync
